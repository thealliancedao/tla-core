# docs — static reference

Hand-maintained reference data the crons read (not cron output, rarely changes).

- `epoch_1-300_date.json` — TLA epoch schedule (epoch number → date). Crons use this for epoch timing.
- `staking-apr.csv` — historical staking APR reference.
- `curated/` — human-curated registry: known contracts, protocols, token overrides, wallets.
